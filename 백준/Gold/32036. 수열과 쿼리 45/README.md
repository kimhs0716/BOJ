### 문제 설명

<p>길이가 $2 \times 10^9 + 1$ 인 배열 $A$ 가 있다. $A$ 의 인덱스는 $[-10^9, 10^9]$ 범위의 정수이다. 초기 $A$ 의 모든 원소는 $0$ 이다. (즉, $A[-10^9] = A[-10^9+1] = \ldots = A[10^9] = 0$)</p>

<p>이 배열에 $q$ 개의 쿼리가 주어진다. 쿼리는 다음과 같은 두 종류이다:</p>

<ul>
	<li><code>1 x y</code>: 모든 $-10^9 \le i \le 10^9$ 에 대해 $A[i] = A[i] + |i - x| + y$ 를 대입한다.</li>
	<li><code>2</code>: $A$ 의 최솟값이 등장하는 가장 첫 지점 $m$와, $A[m]$ 를 출력한다. (첫 지점이라 함은 인덱스가 최소인 지점을 뜻한다.)</li>
</ul>

### 입력 

 <p>첫 번째 줄에 정수 $q$ 가 주어진다. ($1 \le q \le 2 \times 10^5$)</p>

<p>이후 $q$ 개의 줄에 상술한 형태의 쿼리가 주어진다. ($-10^9 \le a, b \le 10^9$)</p>

<p>첫 번째 쿼리는 <code>1 x y </code>형태이다.</p>

### 출력 

 <p>모든 <code>2</code> 형태의 쿼리에 대해서, $A$ 의 최솟값이 등장하는 가장 첫 지점 $m$, 그리고 $A[m]$ 를 공백으로 구분하여 출력하라.</p>
