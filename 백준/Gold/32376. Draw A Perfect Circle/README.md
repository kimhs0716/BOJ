### 문제 설명

<p style="text-align: center;"><img alt="" src="" style="width: 640px; height: 360px;"></p>

<p>동글이는 요즘 동그라미 챌린지를 연습 중입니다. 동그라미 챌린지란 팔을 컴퍼스처럼 휘둘러 칠판에 원을 그리는 놀이입니다.</p>

<p>하지만 동글이는 자신이 그린 그림이 얼마나 원에 가까운지 알기 힘들었고, 당신에게 정확도를 구하는 프로그램을 만들어줄 것을 요청했습니다.</p>

<p>동글이가 그린 그림은 $N$개의 점으로 표현되며, 점을 포함하는 구간의 너비인 $K$가 주어집니다.</p>

<p>정확도는 다음과 같은 기준으로 계산됩니다.</p>

<ul>
	<li>임의의 반지름 $R$에 대해, $cnt(R)$를 원점으로부터의 거리가 $\max(0,R-\frac{K}{2})$ 이상 $R+\frac{K}{2}$ 이하인 점들의 개수로 정의합니다. 	($x$좌표의 차이가 $p$이고 $y$좌표의 차이가 $q$인 두 점의 거리는 $\sqrt{p^2+q^2}$ 으로 정의합니다.)</li>
	<li>이때 정확도는 실수 $R$에 대해서 $\frac{100}{N}\times \max\limits_{R \ge 0} cnt(R)$ 입니다.</li>
</ul>

<p>동글이가 완벽한 원을 그릴 수 있도록 도와주세요!</p>

### 입력 

 <p>첫 번째 줄에 점의 개수 $N(1\leq N \leq 200\,000)$과 너비 $K(1\leq K \leq 20\,000)$가 공백으로 구분되어 주어집니다. $N, K$는 모두 정수입니다.</p>

<p>두 번째 줄부터 $N$개의 줄에 걸쳐 $i$번째 점의 좌표를 나타내는 두 정수 $x_i, y_i$가 주어집니다. $(-10\,000≤x_i, y_i≤10\,000)$</p>

<p>모든 점의 위치는 서로 다름이 보장됩니다.</p>

### 출력 

 <p>동글이가 그린 그림의 정확도를 출력해주세요. </p>

<p>출력한 답과 정답의 절대/상대 오차가 $10^{-6}$ 이하이면 정답으로 인정됩니다.</p>
