### 문제 설명

<p>In what seems to be a familiar occurrence, Farmer John is lining up his $N$ cows ($1\le N\le 10^5$), conveniently numbered $1\ldots N$, for a photograph.</p>

<p>Initially, the cows are lined up in the order $a_1,a_2,\ldots,a_N$ from left to right. Farmer John's goal is to line up the cows in the order $b_1,\ldots,b_N$ from left to right. To accomplish this, he may perform a series of modifications to the ordering. Each modification consists of choosing a single cow and moving it some number of positions to the left.</p>

<p>Please count the minimum number of modifications required in order for Farmer John to line up his cows in the desired order.</p>

### 입력 

 The first line of input contains $N$.  The second line contains $a_1,a_2,\ldots,a_N$.
The third line contains $b_1,b_2,\ldots,b_N$.

### 출력 

 <p>Print the minimum number of modifications required to produce Farmer John's desired ordering.</p>
