### 문제 설명

<p><a href="/user/foxpython">세진(foxpython)</a>이는 역사 속으로 사라져 버린 MatKor Cup을 부활 시키려는 악랄한 계획을 가지고 있다.</p>

<p>기하 마스터인 세진이는 과거 MatKor Cup 문제를 보다가 <a href="/problem/27302">계란을 떨어뜨리면?</a>문제와 <a href="/problem/27303">계란으로 돈을 벌면?</a>문제에 영감을 받아 새로운 문제를 만들었다.</p>

<p>그러나 아직 MatKor Cup을 부활시키기에는 이르다고 판단한 세진이는 KCPC 2025 예비소집으로 이 문제를 보내버렸다.</p>

<p>세진이는 각각 반지름이 $a$, $b$인 원 모양의 두 계란 $A$, $B$를 가지고 있다. 세진이는 또 다른 원 모양의 계란 $C$를 준비해 <strong>세에그트리</strong>를 만들고자 한다.</p>

<ul>
<li>먼저 수직선 모양의 매끈매끈하고 푱푱한 바닥에 계란 $A$와 $B$를 서로 접하도록 놓는다. 이때, $A$와 $B$의 중심을 끝점으로 하는 선분을 $l_{AB}:=\overline{O_AO_B}$라고 하자.</li>
<li>계란 $C$를 바닥에 놓인 두 계란과 모두 접하도록 다음 조건을 만족하게 올려놓을 때 <strong>떨어지지 않는다</strong>고 정의한다.
<ul>
<li>$C$의 중심에서 바닥을 향해 내린 수선을 $l_C$라 할 때, $l_{AB}$와 $l_C$는 한 점에서 만나야 한다. 이 때, 선분의 끝점에서 만나도 된다.</li>
</ul>
</li>
</ul>

<p style="text-align: center;"><img alt="" src="https://upload.acmicpc.net/5bd20f39-108e-4db2-9aea-9172982b6259/-/crop/1426x1630/0,0/-/preview/" style="width: 400px; max-width: 100%;"></p>

<p>세진이는 가능한 큰 계란 $C$를 준비해 <strong>떨어지지 않도록</strong> 하고 싶다. 계란 $C$의 반지름을 $c$라 할 때, <strong>떨어지지 않는</strong> 반지름 $c$의 최댓값을 구해보자. 계란의 반지름은 $0$보다 큰 실수여야 한다.</p>

### 입력 

 <p>첫 번째 줄에 계란 $A$와 $B$의 반지름을 나타내는 정수 $a,b(1\le a,b\le 1\, 000)$가 공백으로 구분되어 주어진다.</p>

### 출력 

 <p>첫 번째 줄에 정답을 다음과 같이 출력한다.</p>

<ul>
<li>반지름이 $0$보다 큰 어떠한 계란도 떨어지지 않는다면 <span style="color:#e74c3c;"><code>INF</code></span>를 출력한다.</li>
<li>위의 경우를 제외하고 $c$의 최댓값이 존재하지 않는다면 <span style="color:#e74c3c;"><code>NO</code></span>를 출력한다.</li>
<li>$c$의 최댓값이 무리수라면 그 값을 출력한다. 이 경우 정답과의 절대/상대 오차는 $10^{-6}$까지 허용한다.</li>
<li>$c$의 최댓값이 유리수라면 그 값을 $\frac{p}{q} (p,q>0$; $\gcd\left( p,q \right) =1)$로 나타낼 때, 정수 $p$와 $q$를 공백으로 구분하여 출력한다.</li>
</ul>
