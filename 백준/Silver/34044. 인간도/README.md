### 문제 설명

<p>인간도의 호반우들은 가장 좋아하는 소수인 $11$을 제외하고 $N$개의 두 자릿수 소수만을 사용하여 $11$의 배수를 만들어야 한다.</p>

<p>정확히는 $N$개의 두 자릿수 소수를 차례대로 이어 붙여 $11$의 배수인 $2N$자리 수를 만들어야 한다. 같은 소수를 여러 번 사용해도 된다.</p>

<p>호반우를 도와 인간도에서 깨달음을 얻어보자.</p>

### 입력 

 <p>첫째 줄에 $N$이 주어진다. $(1 \leq N \leq 100\,000)$</p>

### 출력 

 <p>만약 $N$개의 두 자릿수 소수를 사용하여 $11$의 배수를 만들 수 없다면 첫째 줄에 <span style="color:#e74c3c;"><code>-1</code></span>을 출력한다.</p>

<p>그렇지 않다면 첫째 줄에 두 자릿수 소수를 차례대로 이어붙여 만든 $2N$자리 수를 출력한다.</p>

<p>가능한 방법이 여러 가지라면 그중 아무거나 출력한다.</p>
